Instructions


DashCamShare is a mobile friendly web application for sharing dash camera footage of road accidents that can be used as an evidence for insurance companies.

Go to the home page, find the place where the accident happened on the map. Check for existing tags left by other users, because they can contain useful information, then you can create a tag where the accident happened, click the tag to activate the chat window and either leave a message or upload your video.

While the application by default shaws tags added during last 30 days, you can easily search for any day you like and see what tags were added on that day.
